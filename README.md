Project name :Smart Mess Management System



Description :In most hostels, food is prepared based on rough estimates rather than actual demand. This

leads to:

Excess food preparation

High food wastage

Inefficient use of resources

Increased operational costs

Environmental impact due to wasted food

Therefore, there is a need for a digital system that helps predict meal attendance and optimize

food preparation.
